Personal Reflection

The overall design of my website is minimalistic. It contain the background image at the top part of the web page and the main content underneath the background. I add a smooth transition for the scrolling effect. A logo on the top left corner which can link back to the home page; a top menu banner for desktop users to navigate easily to different web pages; a side navigation menu to help mobile users easier to navigate between pages. During the process, I struggle a lot in finding the correct command to create the effect I wanted. And the most difficult part for me is to figure out how to modify the webpage for mobile devices as I didn't know whhich command to use. I also used different fonts to make the overall web page looks more aesthetic.

Technical Aspects

In the CSS file I used a: hover for sliding effect when the cursor pointing to the 'Read more' area to create a smooth transformation effect. I created this effect by using the smooth scroll behaviour effect and inserting a hyperlink to the #about area of the web page. Moreover, I used this effect in the Future.html webpage as well. When you point to the images, it will show text up with the hover effect. Also, I used the opacity and transition time to create a fade-in hover effect which is more aesthetic and make the transition smoother.

For the background photo, I used the linear gradient command to create a gradient style effect on the photo background to make the color more coherent to each other and make the white-colored text contrast with a darker color background.

Another function I created is the side navigation tool which is for mobile users to easily navigate to different web pages. As it is annoying for users to scroll up to the top banner to click the link to different web pages, the side navigation tap with the fixed position command can help make the task easier. I used the document.getElementById to make the side menu logo works by linking the id of the menu into it. Moreover, I used the menuBut.onclick plus the if/else commands to create two effect. One effect is a clicking effect which the logo will change to another image when the user click on it, and the other one is making the side menu dissapear to the right side of the web page.

Aesthetics

Width of the Webpage

For all the web pages, I make the width 80% and align to centre so all the content can be centre in the middle which is more aesthetic and easy to read.

Title Spacing

In the titles of all web pages, the fonts are bigger and have margin spacing from the main content for a more beautiful outlook.

Hover Effect

I added some hover effects on different areas of the web page to make it more aesthetic. For example, when you point at the 'Read More' button a sliding effect will trigger, and the photos in the future web page as well, hover texts effect will appear when the cursor pointed to the image.

Color Changes of Different Links

The hyperlink in the web pages is all colored differently. For the unvisited links, the color is Medium Aquamarine, visited links are cyan, hot pink when the mouse cursor hovers over it and selected links are red. The colors are contrasted for users to easily identify different meanings of it.

Logo

The logo on the top left corner is both aesthetic and useful as it is linked to the home page. So whenever the user wanted to go back to the home page, they just need to press the logo.

Color Coherence

All the colors and designs on the web pages are very coherent. Besides aesthetic purpose, it can make the user feel more comfortable and feels like they are on the same web page. If the color schemes are so different from page to page, it will make the user confuse whether they are on the same website or going to a different website.

Accessibility Aspects

Color Contrasting

With the help of the Windows 10 greyscale function, I customised the website color to be color contrasted (e.g black and white colors). It can help visually impaired users including color blindness problems easily navigate from the website as contrasting colors on the opposite ends of the color spectrum work best for color-blind users.

Clear and Simple Design

The clear and simplistic design is one of the guidance from WCAG 1.0, which can help users easy to navigate to different parts of the website they desired to go to without going through all the unwanted information. Suitable for ADHD people.

Smooth Transformation

Smooth transformation is also one of the WCAG guidance. And in my website design, there is a 'Read More' button which when you click on it, and the website will have a smooth transformation scrolling down to the main content.
